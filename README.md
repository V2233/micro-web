# Micro-Web

Mciro-Plugin 前端源码

## 安装

- cd 到 micro-plugin 根目录，执行以下命令：

```sh
git clone https://github.com/V2233/micro-web.git frontend
```
或者使用gitee镜像：
```sh
git clone https://gitee.com/V2233/micro-web.git frontend
```

- 安装 yarn

```sh
npm i yarn -g
```

- 安装依赖
  
```sh
yarn install
```

- 启动开发服务器

```sh
yarn run dev
```

- 编译

```sh
yarn run build
```
